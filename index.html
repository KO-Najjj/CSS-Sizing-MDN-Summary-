Task 1: Percentage-based Padding & Margin

Percentages for margin/padding are based on the parent’s width (even vertically).

I’m the child. My margin-top is 10% and padding is 5% of the parent’s width.
  1. In CSS, change .t1-parent { width: 400px; } → 800px.
  2. Observe: margin-top: 10% doubles (40px → 80px). padding: 5% doubles too.

Task 2: Viewport Units (vw/vh)

Resize the browser window—this box and its text scale with the viewport.

Viewport!
  • 50vw = 50% of window width (not parent).
  • 30vh = 30% of window height.
  • 5vh font-size grows/shrinks with height.

Task 3: em vs rem

Outer text (parent). Current font-size: 20px (change to 10px later).

Inner text uses font-size: 1.5em (relative to parent), and margin: 2rem (relative to root).
  1. Note the inner text size now (1.5 × 20px = 30px).
  2. Change .t3-outer { font-size: 10px; }. Inner text becomes 15px, but the 2rem margin stays 32px.
